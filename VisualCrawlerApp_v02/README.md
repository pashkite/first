# Visual Crawler App v0.4

안드로이드 WebView 기반 텍스트 전용 추출 프로토타입입니다.

## v0.4 변경점

- `?` 도움말 버튼을 추가했습니다.
- 목록 페이지에서 카드/제목/링크글/날짜/요약을 선택할 수 있습니다.
- 실제 게시글 내부 페이지로 들어간 뒤 `본문 선택`으로 본문 영역을 따로 지정할 수 있습니다.
- 추출 시 목록의 링크를 하나씩 열어 선택한 본문 영역의 사용자 표시 텍스트만 가져옵니다.
- 결과 저장은 Android Download/다운로드 폴더에 `.txt`로 저장합니다.
- 결과에는 HTML 태그, selector, href 주소를 넣지 않습니다.

## 기본 사용 흐름

1. 블로그 목록 접속
2. 카드 선택
3. 카드 내부에서 제목/링크글/날짜/요약 선택
4. 게시글 하나를 직접 열기
5. 본문 선택
6. 목록으로 돌아와 추출
7. 다운로드 저장

## 주의

사용자가 정상적으로 볼 수 있는 화면의 텍스트를 정리하는 실험용 앱입니다.
캡차 자동 풀이, 차단 회피, 로그인 우회, 유료 콘텐츠 대량 수집 용도가 아닙니다.
