# Visual Crawler App

블로그 목록에서 게시글 카드 하나를 터치하면 비슷한 게시글 영역들을 추출하는 Android WebView 앱 프로토타입입니다.

## 기능

- 앱 안에서 사이트 접속
- 직접 로그인/캡차 해결 가능
- 선택 모드에서 게시글 카드 하나 터치
- 반복되는 카드 영역 자동 감지
- 제목, 링크, 날짜, 미리보기 추출
- 결과를 JSON으로 표시

## APK

GitHub Actions가 `app-debug.apk`를 빌드해서 artifact로 올립니다.

## 주의

이 앱은 사용자가 정상적으로 볼 수 있는 화면의 DOM을 추출하는 실험용 브라우저입니다.
캡차 자동 풀이, 차단 회피, 로그인 우회, 유료 콘텐츠 대량 수집 목적으로 만들지 않았습니다.
